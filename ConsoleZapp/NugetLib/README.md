# ConsoleZapp
A lightweight C# toolkit for console apps — formatted/colored printing, structured lines & headlines, reusable Controls (progress bars, labeled values, percentages...) and a small Tui engine for fixed-header, scrolling console screens.

.NET Standard 2.0 compatible (.NET Framework 4.6.1+, .NET Core 2.0+, .NET 5+).

MIT licensed — use it however you like.

Full docs, roadmap and usage examples: https://github.com/vosiz/consolezapp
